## Features

- Authentication with GitHub
- Dynamic routes
- Internationalization
- Dark mode
- UnoCSS

## Get Started

### Start Supabase

```bash
pnpm dlx supabase start
```

### Fill in the environment variables given by Supabase

```bash
cp .env.example .env
```

### Start the development server

```bash
pnpm dev
```
